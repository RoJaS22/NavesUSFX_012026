Clase Enemigo:
Se creo la clase enemigo a partir de un Actor, se le asigno una malla usando punteros con el componente StaticMeshComponent.
Se creo una función para generar los puntos de ruta que el enemigo va a seguir, siendo estos aleatorios dentro de un rango establecido.
Se creo otra funcion para que el enemigo se mueve por esos puntos de ruta ya definidos.
